# glaucoma-analysis
A data analysis project exploring structural OCT metrics (RNFL, GCC), age groups, and IOP using sample glaucoma data. This repository demonstrates skills in data cleaning, exploratory analysis, visualization, and clinical interpretation using google sheets. 
## Repository Structure

data/
  raw/
    glaucoma_dataset.csv          # Original Kaggle dataset
  processed/
    Final Glaucoma_Data.xlsx      # Cleaned and engineered dataset

docs/
  README Glaucoma.pdf             # PDF version of the project report

README.md                         # Main project documentation
